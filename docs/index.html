Prospective hindsight, made operational

Imagine failure. Then prevent it.

premortem is a structured CLI workflow for finding plausible failure paths, mapping their causes, and turning uncertainty into mitigations and research.

Get started View source
The method

A disciplined path from imagined failure to better decisions.

The workflow uses stakeholder perspectives and causal reasoning—not a generic risk checklist. Explicit review points keep the analysis legible.

01

Define failure

Write a vivid, completed-fact statement focused on consequences rather than causes.

02

Broaden the view

Use distinct stakeholder personas to surface operational, customer, technical, and strategic failure stories.

03

Trace the system

Turn reasons into a compact causal graph showing where root causes converge and amplify one another.

04

Score what matters

Evaluate important nodes by severity, controllability, uncertainty, and available evidence.

05

Break the pathways

Attach concrete, owned, time-bound mitigations to the graph nodes they are designed to change.

06

Test assumptions

Convert unresolved uncertainty into a focused research agenda before the final commitment.

Quickstart

Run the workflow from your terminal.

# Install the package in development mode
python -m pip install -e .

# Start with a concrete imagined failure
premortem init product_launch \
  --failure "It is six months after launch. Adoption is below 10%, support costs have doubled, and key customers have left."

# Check state and move through the analysis
premortem status
premortem analyze personas
premortem analyze reasons
premortem graph build
premortem score nodes
premortem analyze mitigations
premortem analyze research
premortem report generate
The output

A decision aid, not a prediction.

  • Plausible, stakeholder-specific failure mechanisms
  • A readable causal graph of dangerous pathways
  • Concrete mitigations tied to graph nodes
  • Research questions that can change the decision
  • A final report with evidence and recommendations
