# Reporting

A complete report should explain the initiative, the failure statement, the
agent panel, the main causal pathways, mitigations, research agenda, and
limitations.

Include:

- A structured executive summary.
- Persona table.
- Agent panel/model transparency.
- Verbatim excerpts from synthetic persona outputs.
- Causal graph image or clear graph summary.
- Ranked mitigations.
- Research agenda.
- Limitations: synthetic personas are plausible perspectives, not real
  stakeholder evidence.

Generate the interactive HTML report with:

`premortem analyze report`
